NusantaraGo - API Rekomendasi Wisata Indonesia

Selamat datang di dokumentasi API NusantaraGo. API ini menyediakan berbagai endpoint untuk mendapatkan rekomendasi wisata di Indonesia.

Daftar Endpoint

1. Daftar Provinsi

GET /api/provinces

Menampilkan daftar seluruh provinsi yang tersedia.

2. Daftar Kategori

GET /api/categories

Menampilkan daftar seluruh kategori tempat wisata yang tersedia.

3. Daftar Tempat Wisata

GET /api/attractions

Menampilkan daftar tempat wisata dengan berbagai filter.

Parameter:

Parameter Deskripsi
category Filter berdasarkan kategori
province Filter berdasarkan provinsi
min_rating Filter rating minimum
q Pencarian berdasarkan nama
limit Batas jumlah hasil (default: 50)

4. Detail Tempat Wisata

GET /api/attraction/{nama}

Menampilkan detail tempat wisata berdasarkan nama.

5. Rekomendasi Berbasis Konten

GET /api/recommendations/content

Mendapatkan rekomendasi tempat wisata berdasarkan kemiripan dengan tempat wisata tertentu.

Parameter:

Parameter Deskripsi
name Nama tempat wisata yang menjadi acuan (wajib)
limit Batas jumlah hasil (default: 10)

6. Rekomendasi Berdasarkan Popularitas

GET /api/recommendations/popularity

Mendapatkan rekomendasi tempat wisata berdasarkan popularitas.

Parameter:

Parameter Deskripsi
category Filter berdasarkan kategori
province Filter berdasarkan provinsi
limit Batas jumlah hasil (default: 10)

7. Rekomendasi Berdasarkan Lokasi

GET /api/recommendations/location

Mendapatkan rekomendasi tempat wisata berdasarkan kedekatan lokasi.

Parameter:

Parameter Deskripsi
lat Latitude lokasi pengguna (wajib)
lon Longitude lokasi pengguna (wajib)
max_distance Jarak maksimum dalam kilometer (default: 50)
limit Batas jumlah hasil (default: 10)

8. Rekomendasi Hybrid

GET /api/recommendations/hybrid

Mendapatkan rekomendasi tempat wisata menggunakan pendekatan hybrid yang menggabungkan berbagai metode rekomendasi.

Parameter:

Parameter Deskripsi
name Nama tempat wisata yang menjadi acuan
lat Latitude lokasi pengguna
lon Longitude lokasi pengguna
category Filter berdasarkan kategori
province Filter berdasarkan provinsi
max_distance Jarak maksimum dalam kilometer (default: 50)
limit Batas jumlah hasil (default: 10)
